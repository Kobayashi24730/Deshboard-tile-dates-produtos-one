🚀 Dashboard Profissional de Produtos e Vendas

Um Dashboard Full-Stack moderno para gerenciamento de produtos, vendas e métricas de negócios.

O projeto foi desenvolvido com React + TypeScript no frontend e Node.js + Express + PostgreSQL no backend, utilizando arquitetura escalável e boas práticas de desenvolvimento.

---

🖥️ Preview

Interface moderna com:

- 📊 gráficos interativos
- 📦 gerenciamento de produtos
- 👤 autenticação segura
- 📈 métricas em tempo real

---

🧰 Tecnologias Utilizadas

Frontend

- React 19
- TypeScript
- TanStack Query
- Recharts
- CSS puro (Fluent Design UI)

Backend

- Node.js
- Express
- Drizzle ORM
- PostgreSQL
- JWT Authentication

---

📊 Funcionalidades

✔ Login e autenticação JWT
✔ Dashboard com métricas
✔ CRUD completo de produtos
✔ Gráficos de vendas
✔ Tabela dinâmica de produtos
✔ Paginação e busca
✔ Interface responsiva

📦 Instalação

1️⃣ Clonar repositório

git clone https://github.com/seuusuario/dashboard-produtos

---

2️⃣ Instalar dependências

Frontend

cd client
npm install

Backend

cd server
npm install

---

3️⃣ Rodar projeto

Backend

npm run dev

Frontend

npm run dev

---

🧱 Arquitetura

O projeto segue uma arquitetura Full-Stack moderna:

Frontend → React + TanStack Query + TypeScript
Backend → Node.js + Express + TypeScript
Database → PostgreSQL + Drizzle ORM
